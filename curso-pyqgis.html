Curso Práctico

Introducción a
Python en QGIS

Automatización básica de procesos geoespaciales. Este curso está diseñado para profesionales que utilizan QGIS en su día a día y desean aprender a automatizar tareas repetitivas, gestionar bases de datos pesadas y optimizar sus flujos de trabajo utilizando Python (PyQGIS).

Objetivo

Aprender desde la sintaxis básica de programación hasta la creación de scripts funcionales para resolver problemas comunes sin conocimientos previos.

Dirigido a

Arquitectos, biólogos, arqueólogos, urbanistas y profesionales afines a los Sistemas de Información Geográfica (SIG).

Temario Detallado

Módulo 1

Fundamentos de Python desde cero

Objetivo: Perder el miedo al código, entendiendo la sintaxis básica antes de interactuar con el mapa.

Introducción y Primer Contacto

  • ¿Qué es y por qué usar Python en SIG?
  • Uso de la consola de Python integrada en QGIS.

Sintaxis y Estructuras

  • Creación de variables y tipos de datos (texto, números, booleanos).
  • Uso de listas y sentencias condicionales (If / Else).

Buenas prácticas

  • Cómo comentar y organizar el código de forma limpia.
Módulo 2

Introducción a PyQGIS y primeras automatizaciones

Objetivo: Dar el salto a la interfaz de QGIS, aplicando código para consultar información y resolver problemas comunes de coordenadas.

Conectando Python con QGIS

  • Acceso al proyecto actual y selección de capas activas.

Exploración de datos y CRS

  • Consulta de geometrías, campos y atributos mediante código.
  • Sistemas de Coordenadas (CRS): Entendiendo la transformación de EPSG en memoria.
PRÁCTICA ESTRELLA: Extracción automatizada de coordenadas X, Y.
Tomaremos una capa de puntos, transformaremos sus coordenadas al vuelo (ej. de un sistema oficial a otro sistema métrico) y las guardaremos en la tabla de atributos redondeadas a dos decimales.
Módulo 3

Procesamiento geométrico y filtros masivos

Objetivo: Aprender a manipular capas masivas y a realizar cálculos tediosos en cuestión de segundos.

Trabajo con bases de datos pesadas

  • Identificación de atributos en capas oficiales.
PRÁCTICA 1: Automatización de cálculos y uniones.
Uso de código para leer múltiples lotes/polígonos. Cálculo automático de Área y Perímetro en nuevas columnas. Unión masiva (Merge) de todos los lotes procesados en una sola capa final.
PRÁCTICA 2: Filtros de alta velocidad (Datos INEGI).
Carga de una capa masiva vectorial (Ej. Uso de Suelo, Vegetación o DENUE). Aplicación de filtros mediante Python para extraer información específica sin colapsar QGIS.
Módulo 4

Automatización avanzada y cierre de proyecto

Objetivo: Usar las herramientas de geoproceso de QGIS (Processing) directamente desde Python para tareas complejas.

Algoritmos nativos y Cierre

  • Llamado a algoritmos nativos: Cómo usar la Caja de Herramientas desde el código.
  • Cierre del curso: Repaso general, resolución de dudas y recursos para seguir aprendiendo.
PRÁCTICA 1: Interpolación y Recorte (Clip).
Generación de un modelo de interpolación (IDW) a partir de puntos de muestreo y recorte automático del raster resultante utilizando un polígono de límite (Área de estudio).
PRÁCTICA 2 (PROYECTO FINAL): Limpieza y empaquetado de proyectos.
Script para buscar múltiples archivos Shapefiles sueltos en una carpeta y convertirlos automáticamente en una base de datos unificada en formato GeoPackage.

Material del Curso

  • Archivos de práctica para QGIS.

Requisitos

  • Computadora con Windows, Linux o macOS.
  • QGIS versión 3.x instalado.
  • Conexión a Internet estable.
  • No se requieren conocimientos previos de programación.

Resultados de Aprendizaje

  • Comprender la lógica básica de programación con Python.
  • Utilizar la consola Python de QGIS y desarrollar scripts sencillos para automatizar tareas repetitivas.
  • Interactuar con proyectos y capas mediante PyQGIS para consultar información geográfica mediante código.
  • Contar con una base sólida para continuar con cursos de PyQGIS de nivel intermedio y avanzado.
